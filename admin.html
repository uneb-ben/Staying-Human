Connect to GitHub

This is a one-time setup. You'll need a GitHub Personal Access Token so the admin panel can publish to your site automatically.

How to get your token:
1. Go to github.com/settings/tokens/new
2. Give it a name like "My Writing Site"
3. Set expiration to "No expiration"
4. Under Scopes, check repo
5. Click "Generate token" and copy it

Stored only in your browser. Never shared.
